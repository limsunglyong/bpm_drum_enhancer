# 프로젝트 명칭: [Music Beat Generator]

## 현재 목표
- bpm-analyzer-code2.html을 수정하여 왼쪽 화면의 MENU 섹션 부분을 삭제하고 그 영역에 로딩된 MP3 파일의 앨범 아트, 곡정보를 표시하는 기능을 구현해줘.
- 작업이 끝나면 프로그램노트.md 파일의 "변경 이력(Changelog)" 섹션에 파일명과 변경된 내용을 기록해줘.


## 기술 스택
- HTML5, CSS3, JavaScript (Antigravity 환경)

## AI 지시사항 (Prompting)
- 코드를 수정할 때 반드시 이 섹션의 지시를 따를 것.
- UI는 현대적인 디자인을 유지할 것.

## 변경 이력 (Changelog)
- `bpm_analysis/bpm-analyzer-code2.html`: 왼쪽 메뉴(MENU) 섹션을 제거하고, 로딩된 MP3 파일에서 ID3 태그를 추출하여 앨범 아트(Album Art)와 곡 정보(Title, Artist, Album, Year)를 표시하는 UI로 교체함. (`jsmediatags` 라이브러리 연동)
