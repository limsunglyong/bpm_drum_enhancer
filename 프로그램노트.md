- 모든 작업이 완료되었습니다.

- [x] bpm-analyzer-code2.html 파일에서 찾은 BPM으로 Drum beat를 생성해주고 기존의 음원과 믹싱하여 들려주는 기능을 구현함.
- [x] bpm-analyzer-code2.html 메인 화면에서 mp3 waveform 화면 밑에 drum beat waveform을 표시하는 기능을 구현함.

- [x] 기존 mp3 waveform의 zoom, scroll 기능을 drum beat waveform에도 동일하게 적용함.

## 변경 이력 (Changelog)
- `bpm_analysis/bpm-analyzer-code2.html`: 왼쪽 메뉴(MENU) 섹션을 제거하고, 로딩된 MP3 파일에서 ID3 태그를 추출하여 앨범 아트(Album Art)와 곡 정보(Title, Artist, Album, Year)를 표시하는 UI로 교체함. (`jsmediatags` 라이브러리 연동)
- `bpm_analysis/bpm-analyzer-code2.html`: 분석된 BPM에 맞춰 드럼 비트를 합성(Kick/Snare)하고 생성하는 기능을 추가함. MP3 파형 아래에 동기화된 드럼 비트 파형 표시 기능을 구현하였으며, 드럼 믹스 볼륨 조절 슬라이더를 추가함. Zoom 및 Scroll 기능이 두 파형에 모두 동기화되도록 처리함.
- `bpm_analysis/bpm-analyzer-code2.html`: 드럼 비트 타이밍 동기화 기능을 대폭 보강함. 곡 초반 10초를 분석하여 첫 번째 다운비트를 찾는 위상(Phase) 보정 로직을 추가하고, `AudioContext.currentTime`을 이용한 스케줄링 재생 방식을 도입하여 두 소스 간의 정밀한 동기화를 구현함.
- `bpm_analysis/bpm-analyzer-code2.html`: 드럼 비트의 타이밍을 수동으로 미세 조정할 수 있는 **Drum Sync** 슬라이더를 추가함. 사용자가 슬라이더를 조절하면 드럼 파형이 실시간으로 좌우로 밀리며 시각적으로 피드백을 제공하고, 재생 시 해당 오프셋(±200ms)이 정확히 반영되도록 처리함.
- `bpm_analysis/bpm-analyzer-code2.html`: 드럼 사운드(Kick, Snare)를 단일한 **메트로놈 비프음**으로 통합함. 또한, 푸터 레이아웃을 `justify-between`으로 재구성하여 슬라이더와 재생 버튼이 겹치는 UI 문제를 해결함.
- `bpm_analysis/bpm-analyzer-code2.html`: **Sync** 슬라이더의 크기를 2배로 늘리고, 조정 범위를 **-400ms ~ +400ms**로 확대하여 더욱 정밀하고 폭넓은 타이밍 조정이 가능하도록 개선함.
- `bpm_analysis/bpm-analyzer-code2.html`: 템포(BPM) 변경 시 실시간으로 드럼 비트를 재합성하고 재생 중인 오디오에 즉각 반영하는 로직을 구현함. 재생을 멈추지 않고도 변경된 BPM의 메트로놈 소리를 들을 수 있으며, 파형 시각화 또한 즉시 갱신되도록 처리함.
- `bpm_analysis/bpm-analyzer-code2.html`: 킥 드럼 탐지 기반의 정밀 위상 보정 알고리즘을 도입함. 50Hz 로우패스 필터(LPF)로 저역대를 추출하고, 해당 주파수 대역의 에너지 맥동(Pulse)이 설정된 BPM과 동기화되는 최적의 지점을 찾아 메트로놈 시작 시간(Phase)을 자동으로 설정하도록 개선함.
- `bpm_analysis/bpm-analyzer-code2.html`: 파형 줌(Zoom) 기능을 마우스 커서 위치 기반으로 개선함. 확대/축소 시 마우스가 가리키는 파형의 지점이 화면상의 동일한 위치에 유지되도록 오프셋을 자동 보정하여, 더욱 직관적이고 정밀한 탐색이 가능하도록 함.
- `bpm_analysis/bpm-analyzer-code2.html`: 플레이바(Playhead)가 화면 왼쪽 밖으로 벗어났을 때 이를 알려주는 깜빡이는 화살표 오버레이를 추가함. 오버레이를 클릭하면 현재 재생 중인 위치로 즉시 화면이 이동(Scroll back)하는 복구 기능을 포함함.
