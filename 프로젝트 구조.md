# 프로젝트 이름 : Music Beat Generator

## 버전 1.0.0
### 1. 프로그램 개요
* 웹 브라우저 기반의 프로그램
* MP3 파일을 입력하여 음악의 beat를 분석하고 드럼 비트를 추가하는 프로그램

### 2. 프로그램 구성
- **프로그램 구조**
  1. \upload_dashboard\main-code-v6.html : 프로그램 진입점
  - 음악 파일 업로드 대시보드
  - 음악 파일 정보 (곡 정보, Waveform 화면)
  - 음악 재생 기능

  2. \bpm_analysis\bpm-analyzer-code2.html : bpm 분석 코드
  - 피크 찾기 알고리즘
  - 비트 찾기 알고리즘
  - Waveform 그래프 시각화
  - bpm 정보 화면
  - Histogram 그래프 시각화
  - 음악에 맞는 드럼 비트 생성 (Wav)
  - 생성된 드럼 비트와 원본 음악을 합쳐서 재생 기능, MP3로 저장 기능   

  3. \drum_beat_maker\dbm-code.html : Drum BeatMaker 프로그램
  - Advanced 기능
  - 드럼 비트 패턴 추가 및 편집 기능
  - 실시간 드럼 비트 미리듣기 기능
  - 생성된 드럼 비트와 원본 음악을 합쳐서 재생 기능
  - 드럼 비트 저장 기능
  - 자체 드림 비트 포멧 생성 - 파일 저장, 불러오기, 편집 기능

* 작업은 Antigravity를 사용해서 진행할 것.
* Antigravity용 프롬프트를 생성해 줄 것.
